# Água Quente

Máquinas a vapor precisam de vapor para funcionar, e é exatamente isso que a caldeira fornece.

Ela vai usar combustível lentamente para esquentar. Quando estiver quente o suficiente, começará a produzir vapor.

Você também pode fundir baldes de água em baldes de vapor. Construa seu primeiro [Compressor de Bronze](./steam_machines.md) antes da caldeira para economizar muitas chapas!

Vire a página para ver a caldeira em si...

Para fazer tijolos de argila refratária, você precisa assar pós de argila refratária e, para craftá-los, vai precisar de pós de tijolo. Para isso, coloque tijolos no Martelo de Forja. A eficiência do processo pode ser melhorada em máquinas a vapor ou usando um martelo.
