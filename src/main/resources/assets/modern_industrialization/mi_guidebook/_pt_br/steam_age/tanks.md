# Tanques & Barris

Tanques de bronze podem armazenar 4 baldes de fluido cada, você pode usá-los para interagir com máquinas e colocá-los no mundo.

Barris de bronze podem ser usados para armazenar e transportar 32 pilhas de um único item. Confira as próximas páginas para mais informações!

Ao colocar no mundo, você pode clicar com o botão direito usando baldes ou outros tanques para transferir fluidos.

Além disso, você pode empilhá-los no inventário se tiverem o mesmo conteúdo. Que prático!

<Recipe id="modern_industrialization:materials/bronze/craft/tank" />

Clique esquerdo para dar uma pilha para um barril colocado (shift para dar o máximo possível).

Clique direito para pegar uma pilha de um barril colocado (shift para pegar apenas 1 item).

<Recipe id="modern_industrialization:materials/bronze/craft/barrel" />

Se você quebrar um barril, ele manterá o conteúdo armazenado, e você receberá um item com o qual pode interagir.

Clique esquerdo no item barril com um item para inseri-lo no barril.

Clique esquerdo no item barril em um slot vazio para extrair alguns itens dele.
