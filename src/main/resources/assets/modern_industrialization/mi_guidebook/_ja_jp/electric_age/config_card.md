# 高度なパイプ設定

パイプ設定カードはカードパイプをカモフラージュやアイテムパイプの設定のコピーに使える。

<Recipe id="modern_industrialization:tools/config_card" />

このカードをオフハンドに持つと、配置したアイテムパイプに自動的に設定が適用される。
