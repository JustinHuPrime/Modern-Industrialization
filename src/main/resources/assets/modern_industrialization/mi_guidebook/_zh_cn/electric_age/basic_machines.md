# 电力机器

电力机器的工作原理与蒸汽机器完全相同，不同之处在于它们需要电力而不是蒸汽，并且它们会逐步超频配方，最高可达32EU/t。

你可以使用REI查看你已经熟悉的机器的电力版本的配方。

电磁偏振器可以磁化顺磁体。它还没有很多配方，但对于自动化生产电动机已经非常有用！
